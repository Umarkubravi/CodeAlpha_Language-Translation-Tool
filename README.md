# Language Translation Tool

## Overview

Language Translation Tool is a responsive web application built using HTML, CSS, and JavaScript. It allows users to translate text between multiple languages using the MyMemory Translation API.

## Features

- Translate text between multiple languages
- Supports English, Hindi, Urdu, French, Spanish, and German
- Copy translated text with one click
- Responsive user interface
- Clean and modern design
- API-based real-time translation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- MyMemory Translation API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Enter text, select languages, and click **Translate**.
